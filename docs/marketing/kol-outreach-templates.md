## 合作谈判要点

### 我们提供
- 定制股票八字卡片（3-5张）
- 合规文案模板
- 粉丝专属福利码
- 内容审核支持

### 我们要求
1. 必须使用提供的合规文案或经过审核的文案
2. 必须包含免责声明
3. 禁止任何价格预测或买卖建议
4. 发布前需提交草稿审核

### 合作形式
- 小红书图文帖 1 篇
- 或抖音短视频 1 条
- 标注 @市相 或提及品牌

---

## 合规检查清单（发给KOL）

**发布前必须确认：**

- [ ] 包含免责声明："仅供娱乐，不构成投资建议"
- [ ] 无价格预测（"会涨/会跌"）
- [ ] 无买卖建议（"建议买入/卖出"）
- [ ] 无收益承诺（"保证赚钱"）
- [ ] 使用文化/娱乐角度描述
- [ ] 草稿已提交审核并通过

```

7. 发布后跟踪数据
```
