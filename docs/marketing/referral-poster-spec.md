## 设计工具

### Canva 模板 (推荐)
1. 搜索 "dark gradient poster"
2. 选择深色模板
3. 替换文字和 QR 码
4. 导出为 PNG

### Figma
- 使用 Auto Layout
- 确保文字可编辑
- 导出 2x 分辨率

## 海报变体

### 变体1: 通用邀请
- 适用于朋友圈分享
- 强调"早期用户福利"

### 变体2: 股票特定
- 展示热门股票八字卡片
- 吸引对特定股票感兴趣的用户
- 例如：茅台八字海报、特斯拉八字海报

### 变体3: 节日/热点
- 结合当前热点
- 例如：财报季、IPO 热点

## 追踪方法

由于公众号追踪能力有限，建议：

1. **手动统计**
   - 记录每日新增关注数
   - 询问新用户"从哪里知道我们的"

2. **分渠道海报** (可选)
   - 不同渠道使用不同样式海报
   - 根据海报样式判断来源

3. **飞书表格追踪**
   ```
   | 日期 | 新增关注 | 推测来源 | 备注 |
   |------|----------|----------|------|
   ```
